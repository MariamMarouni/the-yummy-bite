# The Yummy Bite 
##  Project Description
This is a responsive ReactJS frontend website for a cake shop called "The Yummy Bite".  
It allows users to browse different cakes with prices and view multiple pages such as Home, Menu, About, and Contact.
##  Technologies Used
- ReactJS
- React Router DOM
- Bootstrap
- CSS
-Git and GitHub
##  Features
- Responsive design (mobile and desktop)
- Navigation between pages
- Cake menu with images and prices
- Contact section with email and phone links
##  How to Run the Project
1. Install dependencies:

npm install

2.Run The Project:

npm start

3.Build Project:

npm run build
